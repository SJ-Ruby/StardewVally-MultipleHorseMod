# Multiple Horse

星露谷物语 1.6 / SMAPI 4 的联机马厩管理模组。

- 在已建造的马厩旁按 `F8` 打开自适应的原生风格菜单。
- 任意玩家可购买额外马匹；主机负责验证马厩、余额及数量上限并创建马匹。
- 额外马免费购买；额外马数不超过当前农场玩家数减一（并受 `MaximumExtraHorses` 限制）。
- 新马在马厩右侧生成。它们带有 `MultipleHorse_childN` 的独立存档名和模组标记，不会触碰原版马厩的马匹数据。
- 购买时使用游戏原生 `NamingMenu`；显示名同时写入马匹的同步数据和 `modData`，可跨过夜与重启保存。

所有联机玩家都应安装同版本模组。首次运行会生成 `config.json`，可修改按键、价格与额外马匹硬上限。
